# English knitting abbreviations (Craft Yarn Council)

Source: craftyarncouncil.com/standards/knitting-abbreviations.

| Abbr | Meaning |
|------|---------|
| CO | cast on |
| BO | bind off / cast off |
| k | knit |
| p | purl |
| st / sts | stitch / stitches |
| k2tog | knit 2 together (right-leaning dec) |
| p2tog | purl 2 together |
| ssk | slip, slip, knit (left-leaning dec) |
| sssk | slip, slip, slip, knit (double dec) |
| k3tog | knit 3 together |
| skp / sl1-k1-psso | slip 1, knit 1, pass slipped st over |
| sk2p / s2kp | central double decrease |
| yo | yarn over |
| kfb | knit front and back (inc) |
| pfb | purl front and back |
| M1 / M1L | make one (left-leaning inc) |
| M1R | make one right |
| M1p | make one purlwise |
| sl | slip |
| sl1 wyif / wyib | slip 1 with yarn in front / back |
| pm | place marker |
| sm | slip marker |
| rm | remove marker |
| rep | repeat |
| rnd / rnds | round / rounds |
| RS / WS | right side / wrong side |
| tbl | through back loop |
| tog | together |
| beg | beginning |
| inc / dec | increase / decrease |
| patt | pattern |
| rem | remaining |
